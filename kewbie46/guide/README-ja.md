|PCB|1|1.6mm 厚 [[gerber](../gerber/kewbie46/jlcpcb)]|

|ボトムプレート|1|1.6mm 厚 [[gerber](../gerber/kewbie46/jlcpcb)]|

|キースイッチ|46|Cherry MX Compatible|
